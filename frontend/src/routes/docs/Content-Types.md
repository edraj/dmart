# Content Types
In Dmart, a “Content Type” defines the structure and format of data stored within a an entry.

## Supported Types
- **text**
- **markdown**
- **html**
- **json**
- **image**
- **python**
- **pdf**
- **audio**
- **video**
- **csv**
- **parquet**
- **jsonl**
- **duckdb**
- **sqlite**

## How to set the content type in the request body
under `record.attributes.payload.content_type`
